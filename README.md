# 🌐🗺️📸 REDVISION – Visualizador de Red Eléctrica

Plataforma web interactiva para la **visualización, monitoreo y digitalización de redes eléctricas**, integrando cartografía, avance de levantamientos SIG y mapeo 360°.

🔗 StoryMap del proyecto:  
👉 https://storymaps.arcgis.com/stories/75ba43bb20324b778c5f9a2f41a6da31

---

## Descripción

**REDVISION** es una aplicación web desarrollada que permite visualizar:

- Avance de digitalización de la red
- Circuitos eléctricos
- Infraestructura (líneas, postes)
- Límites geográficos
- Imágenes reales en campo

---

## Funcionalidades principales

### 📊 1. Panel de control
- Visualización de **porcentaje de avance en la digitalización de la red por circuito**
- Barras dinámicas con colores:
  - 🔴 Bajo (<50%)
  - 🟡 Medio (50–80%)
  - 🟢 Alto (>80%)

---

### ▶️ 2. Simulación de avance
- Reproducción temporal del levantamiento SIG
- Visualización progresiva por fecha
- Controles:
  - ▶ Play
  - ⏸ Pause

---

### 🗺️ 3. Capas geográficas
Activación/desactivación de:

- Líneas eléctricas
- Postes
- Límites
- Vista satelital
- Etiquetas de postes

---

### 📸 4. Integración con Imagenes 360°
Visualización de:

- Secuencias de imágenes
- Puntos individuales

Filtros disponibles:

- **Todas**: Imagenes de todos los usuarios
- **Mías**: Imagenes de usuarios agregados y autorizados
- **Off**: Apagar las capas de las imagenes 360°
